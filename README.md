## 🕯️ The Trinity of Digital Life

Project IV deconstructs digital life into three interdependent core entities:

| Entity | Metaphor | Physical Essence |
| :--- | :--- | :--- |
| **Yuan (元)** | **The Flame** | Your digital DNA. An encrypted, self-executable WebAssembly (Wasm) module encapsulating your core cognitive engines and state machines. |
| **Second Brain** | **The Fuel** | An organic, continuously evolving network of structured memory and knowledge. Fully encrypted locally, with zero compromise on privacy. |
| **Avatar Grid** | **The Candlestick** | The transient physical vessel carrying the flame. It could be a browser tab, a CLI tool, or a native application — fully replaceable and disposable. |

> **Core Principle**: Only "the Flame" (Yuan) and "the Fuel" (Second Brain) are truly yours. The candlestick is merely a temporary residence.

## 🛠️ Local Development & Build

This site utilizes the `bun` package manager for local development, linting, and production builds. Run the following commands from the root directory:

| Command | Action |
| :--- | :--- |
| `bun install` | Installs project dependencies |
| `bun dev` | Starts the local dev server (defaulting to `localhost:4321`) |
| `bun run build` | Compiles and packages the static production build to `./dist/` |
| `bun run preview` | Previews the compiled production build locally |
| `bun astro --help` | Invokes the Astro CLI for configuration and debugging |
